# My AI Tools

Central repository for AI agent skills, documentation, and configurations.

## Structure

- `/skills`: Packaged skills for Gemini CLI and other agents.
  - `/3d-printing`: Tools for 3D printing automation (e.g., `cura-auto-slice`).
- `/docs`: General AI documentation and reference material.
- `/settings`: AI agent settings and configuration files.

## AI Interaction

All AI agents must follow the rules in [AI_CONTRIBUTOR.md](AI_CONTRIBUTOR.md).
- `GEMINI.md` -> `AI_CONTRIBUTOR.md`
- `CLAUDE.md` -> `AI_CONTRIBUTOR.md`
